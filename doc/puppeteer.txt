1. nvim-puppeteer                                 *puppeteer-nvim-puppeteer--*

Masterof strings. Automatically convert strings to f-strings or template
strings and back.


FEATURES                                 *puppeteer-nvim-puppeteer---features*

- When typing `{}` in a python string, automatically convert it to an f-string.
- When typing `${}` in a javascript or typescript string, automatically convert it to a template string.
- When _removing_ the `{}` or `${}`, automatically convert it back to a regular string.
- Also works with multi-line strings.
- No configuration needed, just install and you are ready to go.


REQUIREMENTS                         *puppeteer-nvim-puppeteer---requirements*

- nvim 0.9 or higher.
- The respective Treesitter parsers are required: `:TSInstall python javascript typescript`.


INSTALLATION                         *puppeteer-nvim-puppeteer---installation*

>lua
    -- lazy.nvim
    { 
        "chrisgrieser/nvim-puppeteer",
        dependencies = "nvim-treesitter/nvim-treesitter",
        lazy = false, -- plugin lazy-loads itself
    },
    
    -- packer
    use {
        "chrisgrieser/nvim-puppeteer",
        requires = "nvim-treesitter/nvim-treesitter",
    }
<

No configuration or `.setup()` call is needed. The plugin already automatically
loads as little as possible.
